## Union-find
### Time Consuming
|           algorithm           | worst-case time |
| :---------------------------: | :-------------: |
|          quick-find           |       M N       |
|          quick-union          |       M N       |
|          wighted QU           |   N + M logN    |
|     QU + path compression     |   N + M logN    |
| wighted QU + path compression |   N + M lg*N    |

### Application
Percolation Model(渗透模型): Monte Carlo Simulation
